## üÜò Quick Troubleshooting

### Photos Not Showing?
- Make sure photos are in `images/photos/` folder
- Check the filenames are correct (case-sensitive!)
- Hard refresh the browser (Cmd/Ctrl + Shift + R)

### Timeline Not Expanding?
- Make sure JavaScript is enabled in your browser
- Check browser console (F12) for errors
- Try clicking directly on the card (not just the edge)

### Lightbox Not Opening?
- Make sure JavaScript is enabled
- Try hard refreshing the browser
- Check that you're clicking on the photo, not around it

### Navigation Not Working?
- Hard refresh the browser
- Check that you're viewing at http://localhost:8080
- Make sure the server is still running

### Changes Not Appearing?
- Save the file first
- Hard refresh: Cmd/Ctrl + Shift + R
- Clear browser cache if needed
