# Our Story - Birthday Gift Website

A personalized, interactive website celebrating our relationship with photos, timeline, and hidden love letters.

## Project Structure
```
our-story/
├── index.html              # Main page
├── css/
│   ├── style.css          # Main styles
│   └── animations.css     # Animation styles
├── js/
│   ├── main.js            # Core functionality
│   ├── slideshow.js       # Photo slideshow logic
│   ├── timeline.js        # Timeline interactions
│   ├── love-letters.js    # Hidden letter triggers
│   └── password.js        # Password protection
├── images/
│   ├── photos/            # All relationship photos
│   ├── timeline/          # Timeline milestone images
│   └── backgrounds/       # Background images
├── audio/
│   └── our-song.mp3       # Background music
└── data/
    ├── timeline-data.js   # Timeline content
    └── letters-data.js    # Love letter messages
```

## Setup Instructions

1. Add your photos to `images/photos/`
2. Add your special song to `audio/our-song.mp3`
3. Fill in your timeline stories in `data/timeline-data.js`
4. Write your love letters in `data/letters-data.js`
5. Open `index.html` in a browser to test
6. Deploy to GitHub Pages when ready

## Libraries Used
- Swiper.js - Photo slideshow
- AOS - Scroll animations
- PhotoSwipe - Gallery lightbox
- Leaflet.js - Interactive map (optional)

## Development
Open this folder in VS Code and use GitHub Copilot to build each feature step by step.

## Deployment
Push to GitHub and enable Pages in repository settings.
