## üÜò Troubleshooting

### White screen or nothing showing?
- Check server is running in terminal
- Try http://127.0.0.1:8080 instead
- Hard refresh: Cmd/Ctrl + Shift + R

### Timeline not expanding when clicked?
- Hard refresh the browser
- Check browser console (F12) for errors
- Make sure JavaScript is enabled

### Photos not showing?
- Check photos are in `images/photos/` folder
- Verify filenames match (case-sensitive!)
- Hard refresh browser

### Lightbox not opening?
- Make sure you're clicking on the photo itself
- Hard refresh browser
- Check browser console for errors

### Changes not appearing?
- Save the file
- Hard refresh (Cmd/Ctrl + Shift + R)
- Clear browser cache if needed
